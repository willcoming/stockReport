多空辯論

多方論點： GOOGL 的核心 thesis 是「AI 不是侵蝕搜尋，而是重估整個 Google stack」。Q1 Search & Other 仍年增 19%，Cloud 年增 63%，backlog 超過 $460B，Gemini API token usage 與 paid subscriptions 均快速成長；加上 TPU hardware agreements 與 Android / Chrome / Workspace 的 distribution，Alphabet 有能力把 AI 模型、晶片、雲、廣告與裝置串成封閉式商業飛輪。Berkshire 加碼也提供機構信心背書。

空方論點： 股價與估值已反映很多好消息，P/FCF 高且 RSI 偏熱；AI capex 從 Q1 $35.7B 拉到全年 $180B-$190B，2027 還要再升，代表自由現金流與折舊壓力會持續。更大的問題是核心 Search ad pool 可能被 AI assistant、Perplexity、OpenAI、Copilot 與監管補救措施分流；若 Google 需要犧牲 ad load 或分享 search data / distribution，市場會重新折價。

關鍵分歧：

  • AI Overviews / Gemini / Android agent 能否增加 Search query 與商業轉換，而不是降低傳統點擊與廣告庫存。
  • Cloud TPU 與 enterprise AI backlog 能否以高毛利變現，抵消 capex、折舊、Wiz 整合與資料中心承諾。
  • Google I/O 2026 是否能讓市場相信 GOOGL 在 agentic AI 的產品落地速度領先，而不是只是在防守 OpenAI / Perplexity / Microsoft。

風險管理員備註

  • APPROVES WITH ADJUSTMENT（調整後通過）： Trader 的 HOLD 結論合理，但不應在 I/O 前後追高；建議等待事件後確認或等待回測支撐。
  • 主要 downside scenarios：I/O 發表低於預期、Gemini 4.0 或 agent roadmap 未獲市場認可、AI capex guidance 再上修、Cloud margin 被 Wiz / TPU hardware mix 稀釋、Search antitrust 補救措施影響 distribution。
  • Position-sizing：若作為單一大型科技股 exposure，需控制集中度；GOOGL 與 MSFT、AMZN、META、NVDA 同屬 AI / mega-cap risk bucket，相關性高。
  • Liquidity concern 低：GOOGL 成交量與市值足夠；真正風險不是出場流動性，而是事件風險造成 gap down。
  • 風控調整：只有在 $400-$405 上方放量站穩、或 $365-$385 回測不破後，才把 HOLD 轉為更積極的偏多；若跌破 50 日均線附近並伴隨 capex / regulatory 負面消息，降級為防守觀望。
