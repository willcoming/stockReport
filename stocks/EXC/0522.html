多空辯論

多方論點： EXC 是 fully regulated transmission / distribution utility，Q1 2026 營收成長約 7.9%，公司維持 $2.81-$2.91 FY2026 EPS guidance 與 2025-2029 接近 5%-7% 高端的 EPS CAGR；$41.7B 四年 capex 與 7.9% rate base growth 支撐中期可見度。估值 forward P/E 15.99 低於 DUK 的 18.66 與 Utilities sector P/E 22.04，且 dividend yield 約 3.63%，提供防禦性 total return cushion。最大空方反駁是：這些成長依賴大量資本支出、債務與監管回收，若融資成本或客戶電價壓力惡化，EPS 成長會被稀釋。

空方論點： EXC 的 Q1 adjusted EPS 從 $0.92 降到 $0.91，營業利益率年減；資產負債表高槓桿，Debt/Equity 1.75、Debt/EBITDA 6.17，且公司有 ATM / forward sale equity financing，未來股權稀釋不可忽視。ComEd 與 PJM capacity cost 相關新聞顯示客戶電費與 data center 用電壓力正在升高，這可能引發政治與監管壓力，壓縮 allowed return 或延遲成本回收。最大多方反駁是：EXC 的 regulated model 與 decoupling 機制降低用電量波動風險，且 FERC price collar 可緩解部分 affordability shock。

關鍵分歧：

  • EXC 的 $41.7B capex 是「可回收的 rate base growth」還是「高利率下的槓桿與稀釋壓力」。
  • PJM / ComEd 電費壓力會被 price collar、客戶補助與監管機制吸收，還是轉化為政治風險與 allowed return 壓力。
  • 目前 forward P/E 15.99 是防禦型公用事業折價機會，還是市場合理反映低成長、高資本需求與短線技術壓力。
