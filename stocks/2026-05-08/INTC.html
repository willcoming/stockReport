多空辯論

多方論點：

  • Q1 2026 顯示核心產品需求回溫：revenue 年增 7%、non-GAAP EPS $0.29、non-GAAP gross margin 41.0%，DCAI revenue 年增 22%，支持「AI inference / server CPU / agentic AI 帶動 CPU 需求重估」的 thesis。
  • Google 官方 multiyear collaboration 強化 CPU / IPU 在 AI infrastructure 的角色；每日報告上下文提到 TPU / 18A / 14A 與本土 foundry 戰略價值，和公開新聞方向一致。
  • Foundry 雖虧損，但 18A ramp、advanced packaging、Apple / Google / Nvidia 相關傳聞讓市場開始賦予「美國本土先進製程替代鏈」期權價值。多方承認最大反方是：這些客戶多數仍未轉化為大額、可驗證、可獲利的外部營收。

空方論點：

  • 現價已大幅提前反映成功轉型：Forward PE 約 105.85，TTM EPS 仍為負，分析師共識仍是 Hold，平均目標價 $65.44，與現價落差大。
  • Foundry Q1 operating loss 仍達 $2.437B，external revenue 只有 $174M；如果 18A / 14A 客戶落地慢，核心產品現金流可能長期補貼代工資本支出。
  • 技術面 RSI 85.92 且股價遠高於 50 / 200 日均線，短期 profit-taking 風險高。空方承認最大多方反駁是：若 Apple / Google / Nvidia 類大型客戶確認，市場可能繼續用戰略資產而非傳統 EPS multiple 重新定價 INTC。

關鍵分歧：

  • Foundry 的外部客戶 pipeline 是否會在 2026 下半年轉化為可驗證訂單與 2027 revenue visibility。
  • Q1 的 gross margin 改善是結構性改善，還是 supply / mix / pricing 與一次性因素共同造成的高點。
  • 市場是否應該用「戰略 foundry + AI infrastructure option value」給 INTC 高倍數，或回到 legacy semiconductor earnings multiple。

風險管理員備註

  • 調整後通過 HOLD thesis：不反對保留觀察或既有部位，但不批准在 RSI 過熱、forward multiple 高企、Foundry 虧損尚未明確收斂時追價加碼。
  • Position sizing 應低於一般核心持股，因為 INTC 同時具有高 beta、政策敘事、foundry execution 與 options-driven volatility；避免讓單一轉型 thesis 形成過度集中。
  • Liquidity 不是主要問題；主要問題是 gap risk。Apple / Google / Nvidia 客戶傳聞若被否認，或 Q2 margin / Foundry loss 低於預期，股價可能快速回測技術支撐。
  • 風控調整：等待「財報部門利潤率拆解 + Foundry loss narrowing + 外部客戶合約確認」三者至少兩項成立後，才把 HOLD 上修為 BUY；若跌破 $75.40 並伴隨放量，應重新評估 thesis。
