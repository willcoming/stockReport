多空辯論

多方論點： ServiceNow 不是單純被 AI agent 取代的傳統 SaaS，而是在把自己定位成企業 AI agent 的治理、資料脈絡、流程執行與稽核層。Q1 訂閱收入年增 22%、cRPO 年增 22.5%、非 GAAP 營業利益率 32%，顯示客戶需求與營運槓桿尚未崩壞。公司也把 AI、Armis、Veza、Google Cloud、AWS 等合作與併購串成「AI control tower」敘事，若企業最終需要可治理、可稽核的 agent workflow，NOW 可能是受益者而非受害者。

空方論點： 市場真正擔心的是商業模式重估：AI agent 若降低人員席位、降低前端 SaaS 介面使用頻率，ServiceNow 的高成長與高倍數可能同時承壓。Armis 約 $7.75B 現金併購提高整合風險，也讓 2026 年 margin 指引受到短期稀釋。技術面仍低於 200 日均線，空頭部位上升，代表市場尚未完全接受「AI 是增量而非替代」的論點。

關鍵分歧：

  • AI agent 是壓縮 ServiceNow seat-based SaaS 價值，還是提高其作為企業流程治理層的價值？
  • 2026 年約 22% 訂閱收入成長能否在 Armis 貢獻之外維持有機動能？
  • 股價站回 100 日 / 200 日均線前，基本面強勁是否足以抵消 SaaS disruption 估值壓縮？

  • 若已有部位，風險控制重點是避免單一高估值軟體股集中度過高；若無部位，應等待 $111 附近 100 日均線突破或 Q2 財報確認，而不是只因股價較高點回落就假設便宜。
  • 下行情境包括：AI 收費模式低於預期、Now Assist / Autonomous Workforce 無法轉成高品質 ACV、Armis / Veza 整合拖累 margin、或大型企業 IT 預算轉向更開放 / 更低成本的 agent stack。
  • 流動性不是主要問題，NOW 成交量充足；主要風險是估值與敘事風險，尤其在 1-6 個月期間內，技術面與市場風格可能比基本面更主導股價。
