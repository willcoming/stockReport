多空辯論

多方論點： MSFT 的核心 thesis 是「企業 AI 變現 + Azure 供給受限但需求強」。FY26 Q3 Azure +40%、Microsoft Cloud +29%、AI annual revenue run rate 超過 370 億美元，且 M365 Copilot 已有 2,000 萬 paid enterprise seats，代表 AI 不是單純 demo，而是在企業軟體分發管道內開始商業化。Ackman 建倉也強化「高品質科技股修正後重估」敘事。最大反方論點是 CapEx 過大，但多方認為供給受限本身反映需求超過產能。

空方論點： 最大風險不是營收，而是資本密集化。管理層指引 calendar 2026 CapEx 約 1,900 億美元，Q4 超過 400 億美元，且 Microsoft Cloud gross margin 受 AI 投資與 GitHub Copilot 使用增加拖累。若 OpenAI 關係、模型競爭、電力 / 資料中心瓶頸或企業 Copilot ROI 低於預期，市場可能重新壓低 P/E。最大多方反駁是 MSFT 仍有 24x 左右 TTM P/E 與強現金流，不像純 AI infrastructure play 那樣單一風險暴露。

關鍵分歧：

  • AI CapEx 是高 ROI 產能前置，還是把高毛利軟體公司變成更資本密集的雲端基建公司。
  • Copilot 付費席次與使用量能否轉成可見 ARPU / margin expansion，而不是只提高推論成本。
  • 421-438 美元區間是否能突破並形成 re-rating，或只是 Ackman headline 帶來的短線反彈。
