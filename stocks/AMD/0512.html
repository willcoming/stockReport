多空辯論

多方論點： AMD 的中期核心變化不是單季 beat，而是資料中心從「GPU 替代敘事」擴展到「AI infrastructure CPU + GPU + rack-scale」敘事。Q1 資料中心營收 $5.8B、年增 57%，Q2 指引 $11.2B 且年增約 46%，加上 Meta 6GW、MI450 / Helios、EPYC 雲端 instance 擴張，支持每日報告提到的 agentic AI 與長上下文推升伺服器 CPU 需求。最大反方問題是估值已把大量未來成長提前反映。

空方論點： AMD 已經在短時間內大幅 rerating，2026-05-11 收盤 $458.79，接近 52 週高點 $469.22，RSI 超過 80，TTM P/E 約 153x；若 Q2 或下半年供應鏈、台積電先進製程 / 先進封裝、ABF 載板、HBM / 記憶體成本、遊戲需求任一環節低於預期，股價回撤可以很快。最大多方反駁是 AMD 現金充足、資料中心成長已從敘事變成財報數字。

關鍵分歧：

  • AMD 是否能把 agentic AI 帶來的 CPU 負載轉化為可持續的 EPYC share gain，而不只是一次性財報驚喜。
  • 台積電先進製程、先進封裝、HBM / 記憶體與 ABF 載板供應是否足以支撐 AMD 擴大出貨，而不被 GPU 大客戶排擠。
  • 當股價已接近高位且估值昂貴時，未來 1-6 個月的上修空間是否大於動能反轉與多頭獲利了結風險。

風險管理員備註

  • 結論： APPROVED WITH ADJUSTMENT（調整後通過）。Trader 的 HOLD 合理，但不批准追價式 BUY；調整為「續抱 / 等確認」，並要求較小風險曝險、分批、等待 pullback 或 Q2 驗證。
  • 主要 downside scenario 是「基本面好但不夠好」：Q2 指引已高，若資料中心成長、MI450 / Helios customer forecast、server CPU growth 任一項低於市場高預期，估值壓縮可能主導股價。
  • 供應鏈風險需獨立監控：每日報告指出台積電先進製程產能被 GPU 擠壓、良率與封裝材料短缺；這會影響 AMD 是否能把訂單轉成收入。
  • 技術面不宜忽視：RSI overbought、價格接近 R1/R2，若跌破 $430.48 或 $405.76，代表財報後動能開始失守，需重新評估。
  • 集中度風險：AMD 屬高 beta 半導體 AI exposure；若投資組合已高度集中於 AI / 半導體 / 台積電供應鏈，需降低單一股票加碼衝動。
