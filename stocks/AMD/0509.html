多空辯論

多方論點： AMD 的核心變化是 Data Center 已成為收入與獲利主引擎，Q1 Data Center revenue 年增 57%，Q2 公司指引也顯示 server / AI infrastructure demand 延續。agentic AI 帶動 CPU orchestration、data movement、head-node operation 需求，讓 AMD 不只押 GPU，也受惠於 EPYC CPU 市占提升。若 Meta / OpenAI / cloud customer 採購與 MI450 / MI455X ramp 順利，市場可能繼續上修 2026-2027 EPS 與營收。

空方論點： 股價已把大量未來成長折現進去，TTM P/E 約 153，日線 RSI 超過 84，使用者每日報告也標記周線 RSI 接近歷史極度超買。H2 gaming revenue 可能因記憶體與零組件成本下降超過 20%，而 Meta / OpenAI warrant 結構若達成 milestone 可能帶來潛在稀釋。若 AI server 出貨節奏延後、毛利被 Helios / MI450 ramp 壓縮，估值修正會很劇烈。

關鍵分歧：

  • Data Center CPU / GPU 的加速成長是否能持續到 2026 下半年與 2027，而不是 Q1 / Q2 的一次性 pull-in。
  • AMD 能否在擴張 AI supply、TSMC allocation 與 HBM4 供應時維持或提升毛利率。
  • 市場是否願意在 RSI 極度超買與高 P/E 下繼續給予更高估值，或轉為要求實際 EPS 兌現。

  • 倉位風險： AMD beta 約 1.96，且目前成交量放大、RSI 極高，medium horizon 下不應集中持有單一高 beta AI 硬體股；若已有 NVIDIA、記憶體、AI server 供應鏈曝險，AMD 會增加同一因子集中度。來源：StockAnalysis
  • 流動性風險低，但波動風險高： AMD 交易量充足，退出不是主要問題；主要問題是高估值下的 gap-down 與多頭擁擠。
  • 稀釋 / 客戶集中風險： 10-Q 披露 OpenAI 與 Meta 各有最高 160M 股 warrant，需達成採購 milestone 與股價條件才 vest；這強化客戶合作，但也使長期股本稀釋成為需監控項。來源：AMD 10-Q
  • 主要監控點： Q2 revenue 是否落在或高於 $11.2B ± $300M、non-GAAP gross margin 是否接近 56%、Data Center revenue / operating margin 是否延續、server CPU revenue 是否符合管理層高成長敘事。
