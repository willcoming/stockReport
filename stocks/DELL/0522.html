  • AI server 成長可能是低 margin 成長：年報明確指出 FY2026 gross margin percentage 下滑主因是 AI-optimized server mix，若市場只看營收與 backlog，可能低估利潤率風險。Dell FY2026 Annual Report

  • 空方承認的最大多方論點：AI server backlog 430 億美元與 FY2027 revenue midpoint 1,400 億美元代表需求不是單季題材，若 margin 穩住，市場可能繼續上修合理估值區間。Dell FY2026 Q4 Results PDF

關鍵分歧：

  • AI server 高成長到底能否轉化成穩定 operating margin，而不是只帶來低毛利硬體營收。
