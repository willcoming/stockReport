## 關鍵假設說明

- FCF 成長率來源：FY2025 FCF 為負，無法用 FY2021 到 FY2025 直接計算有效 CAGR；本模型改用 TTM FCF NT$69.33M 對 FY2021 FCF NT$16.24M 的 4 年 CAGR，得到 43.7%。此成長率很高，主要反映低基期與 FCF 波動。
- WACC 組成：無風險利率 4.60% + 模型 Beta 1.00 x 5.5% ERP = 10.10%。報告 Beta 為 -0.14，但若直接套用會使 bull WACC 低於終值成長率，Gordon terminal value 失效，因此模型採用 1.00 作為保守 beta floor。
- 終值佔總估值比重：81.1%（基準情境）。
- 若終值佔比 >70%，結果對終值假設高度敏感，請謹慎解讀。
- 本次 valuation 使用 Bash 執行 Python DCF 計算，projection_years = 5；金額單位為新台幣。
