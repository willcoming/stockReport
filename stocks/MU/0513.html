多空辯論

多方論點： MU 是全球記憶體報價與 AI memory 週期的重要風向標。最新財報顯示營收年增 38%、毛利率明顯修復，且第三季營收指引仍維持高檔。DDR5 server RDIMM、high-capacity SSD、HBM / AI server demand 都支持中期成長敘事。多方承認最大的反方論點是：股價短期已快速反映未來利多，技術面過熱，若下一輪指引未再上修，估值可能先壓縮。

空方論點： MU 是高度循環股，當市場把 AI memory shortage 線性外推到未來數季時，最容易在景氣高點附近給出過高倍數。第二季自由現金流仍為負，資本開支高，若供給擴張或客戶拉貨放緩，毛利率修復可能停滯。空方承認最大的多方反駁是：本輪不是一般 PC / smartphone memory cycle，而是 AI server、HBM、DDR5 共同推動，供給約束可能比過去週期更持久。

  • 目前股價反映的是「合理的 AI memory 重估」，還是「已提前反映未來兩年獲利」。
